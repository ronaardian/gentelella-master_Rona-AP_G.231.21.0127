Cara Penggunaan

Untuk alasan kinerja, semua ikon memerlukan kelas dasar dan kelas ikon individual. Untuk menggunakannya, letakkan kode berikut di mana saja. Pastikan untuk menyisakan ruang antara ikon dan teks untuk padding yang tepat.

Jangan dicampur dengan komponen lain

Kelas ikon tidak dapat digabungkan secara langsung dengan komponen lain. Mereka tidak boleh digunakan bersama dengan kelas lain pada elemen yang sama. Sebagai gantinya, tambahkan <span> bertingkat dan terapkan kelas ikon ke <span> tersebut.

Hanya untuk digunakan pada elemen kosong

Kelas ikon hanya boleh digunakan pada elemen yang tidak berisi konten teks dan tidak memiliki elemen turunan.

Mengubah lokasi font ikon

Bootstrap mengasumsikan file font ikon akan berlokasi di ../fonts/ direktori, relatif terhadap file CSS yang dikompilasi. Memindahkan atau mengganti nama file font tersebut berarti memperbarui CSS dengan salah satu dari tiga cara berikut:

  • Mengubah@icon-font-path and/or @icon-font-name variabel dalam file sumber Lebih sedikit.
  • Memanfaatkan opsi URL relatif disediakan oleh kompiler Less.
  • Mengubah url() jalur dalam CSS yang dikompilasi.

Use whatever option best suits your specific development setup.

Ikon yang dapat diakses

Versi modern dari teknologi bantu akan mengumumkan konten yang dihasilkan CSS, serta karakter Unicode tertentu. Untuk menghindari keluaran yang tidak diinginkan dan membingungkan pada pembaca layar (khususnya bila ikon digunakan semata-mata untuk dekorasi), kami menyembunyikannya dengan aria-hidden="true" atribut.

Jika Anda menggunakan ikon untuk menyampaikan makna (bukan hanya sebagai elemen dekoratif), pastikan bahwa makna ini juga disampaikan ke teknologi bantu misalnya, sertakan konten tambahan, yang secara visual disembunyikan dengan .sr-only class.

Jika Anda membuat kontrol tanpa teks lain (seperti <button> yang hanya berisi ikon), Anda harus selalu menyediakan konten alternatif untuk mengidentifikasi tujuan kontrol, sehingga masuk akal pengguna teknologi bantu. Dalam hal ini, Anda dapat menambahkan aria-label atribut pada kontrol itu sendiri.

Copy 
<span class="glyphicon glyphicon-search" aria-hidden="true"></span>

Contoh

Gunakan di tombol, grup tombol untuk toolbar, navigasi, atau input formulir yang ditambahkan.

Copy
<button type="button" class="btn btn-default" aria-label="Left Align">
  <span class="glyphicon glyphicon-align-left" aria-hidden="true"></span>
</button>

<button type="button" class="btn btn-default btn-lg">
  <span class="glyphicon glyphicon-star" aria-hidden="true"></span> Star
</button>

An icon used in an alert to convey that it's an error message, with additional .sr-only text to convey this hint to users of assistive technologies.

Error: Enter a valid email address
Copy 
<div class="alert alert-danger" role="alert">
  <span class="glyphicon glyphicon-exclamation-sign" aria-hidden="true"></span>
  <span class="sr-only">Error:</span>
  Enter a valid email address
</div>
